# UUON Foundation Budget Allocation Framework

## Core Principle
**100% of earnings → Legal/Trust Setup FIRST**

All revenue flows through this filter before ANY other allocation:

```
Revenue In
    ↓
├─ PHASE 1: Legal/Trust (100% until complete)
│  ├─ Trust attorney (US): €15K-30K
│  ├─ Trust attorney (DE): €10K-20K
│  ├─ Trust establishment: €5K-10K
│  ├─ DMCA/IP protection: €5K-10K
│  ├─ Regulatory pathway (DE): €20K-50K
│  └─ Legal insurance: €5K-10K/year
│  TOTAL PHASE 1: €60K-130K
│
├─ PHASE 2: Infrastructure (after Phase 1)
│  ├─ Server costs: €11-35/month
│  ├─ Domain/SSL: €50/year
│  ├─ Database (Neon): Included in Railway
│  └─ Deployment (Railway): $20-35/month
│
├─ PHASE 3: Operations
│  ├─ Developer bounties: €5K-20K/month
│  ├─ Marketing/growth: €2K-10K/month
│  └─ Community support: €1K-5K/month
│
└─ PHASE 4: Founder Benefits
   ├─ Salary (automated): €100K/month (AFTER phases 1-3)
   ├─ Bonus pool: €20K/month (discretionary)
   └─ Reinvestment: 40% of revenue back to foundation
```

## Revenue Targets (Conservative)
- Month 1-2: €5K-20K (C Bot, mining, initial traction)
- Month 3-6: €50K-200K (multi-chain, Osmosis deal)
- Month 6-12: €500K-2M (data reduction service live)
- Year 2+: €5M-50M (validated product-market fit)

## Enforcement
Smart contract on Base mainnet:
- All earnings flow to Treasury wallet FIRST
- Automatic split: 100% → legal until flag "LEGAL_COMPLETE"
- Then: 50% ops, 30% founder salary, 20% reinvestment
- Transparent on-chain, immutable
